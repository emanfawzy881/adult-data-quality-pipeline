# Adult Data Quality Pipeline

This project implements a simple automated data quality pipeline for the UCI Adult dataset.

## Goal
The goal of this project is to combine data cleaning, rule-based validation, and monitoring in one simple workflow.

## Dataset
- UCI Adult dataset

## Tools
- Python
- pandas
- Great Expectations
- Evidently

## Project Structure
- `data/raw/` raw dataset
- `data/processed/` cleaned dataset
- `src/` source code
- `reports/` generated reports

## Pipeline Steps
1. Load data
2. Clean data
3. Validate data
4. Monitor data
5. Generate outputs
